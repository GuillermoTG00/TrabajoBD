Pasos para ejecutar la app.

1. Instalar wamp o xamp server, ejecutarlo activar los servicios de apache y mysql.
2. Copiar esta carpeta clase a la carpeta www (Wamp) o htdocs (Xampp) correspondiente a la app instalada.
3. Actualizar dentro de la carpeta configuraciones el archivo conexion con los datos de usuario y bd.
4. Se debe ejecutar el script createdb.sql en la base de datos creada.
